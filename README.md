# urban-scanner
Il tessuto urbano è sottoposto a riconfigurazione speculativa. Un "corpo esterno" istituzionale sovrascrive la realtà locale: lo spazio viene amministrato come coordinata di profitto per grandi capitali. Il gesto del cittadino viene tradito: la periferia che chiedeva un futuro per i propri figli riceve invece un futuro per gli investitori.
